# lite-saju (만세력 계산기)

한국 전통 사주명리학 기반의 만세력 계산기 웹 애플리케이션입니다.

## 📖 소개

lite-saju는 생년월일시를 입력하면 사주팔자를 계산하고 다양한 명리학적 정보를 제공하는 웹 도구입니다. 단일 HTML 파일로 구성된 정적 웹앱으로, 별도의 설치나 서버 없이 브라우저에서 바로 실행할 수 있습니다.

### 사주란?

사주(四柱)는 한국과 중국의 전통 명리학으로, 태어난 해·달·일·시를 기준으로 운명과 성격을 분석하는 학문입니다. 각 기둥은 천간(天干) 10개와 지지(地支) 12개의 조합으로 이루어집니다.

## ✨ 주요 기능

- **사주팔자 계산**: 생년월일시 입력으로 년주·월주·일주·시주 계산
- **십성(十星)**: 일간을 중심으로 다른 천간과의 관계 분석
- **십이운성(十二運星)**: 각 기둥의 생명 주기 단계 표시
- **십이신살(十二神殺)**: 길흉을 나타내는 12신장 계산
- **귀인성(貴人星)**: 천을귀인, 천주귀인, 문곡귀인, 태극귀인 등 길성 표시
- **오행 색상**: 목(녹색), 화(빨강), 토(갈색), 금(회색), 수(검정)으로 시각화
- **반응형 디자인**: 모바일과 데스크톱 모두 지원

## 🛠️ 기술 스택

- **순수 HTML/CSS/JavaScript**: 프레임워크 없음
- **무의존성**: 외부 라이브러리 불필요
- **단일 파일**: index.html 하나로 모든 기능 구현
- **빌드 과정 없음**: 수정 후 즉시 실행 가능

## 🚀 실행 방법

1. 이 저장소를 클론하거나 `index.html` 파일을 다운로드합니다.
2. `index.html` 파일을 웹 브라우저에서 직접 열면 됩니다.

```bash
# 저장소 클론
git clone https://github.com/lunara-kim/lite-saju.git
cd lite-saju

# 브라우저에서 열기 (macOS 예시)
open index.html

# 또는 파일을 더블클릭하여 기본 브라우저로 열기
```

별도의 서버나 빌드 과정이 필요하지 않습니다.

## 🎨 오행 색상 매핑

- 목(木) → 녹색
- 화(火) → 빨강
- 토(土) → 갈색
- 금(金) → 회색
- 수(水) → 검정

## 🧮 계산 방법

- **년주**: 음력 입춘(立春) 기준으로 계산
- **월주**: 절기(節氣) 기준으로 계산
- **일주**: 2024-01-01(갑자일)을 기준으로 상대 계산
- **시주**: 전통적인 12시진 체계 사용 (2시간 단위)
